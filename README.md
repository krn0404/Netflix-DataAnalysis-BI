Proyecto de Inteligencia de Negocios - Análisis de Netflix

Proceso ETL: Limpieza y transformación de datos desde Excel hacia SQL Server.
Modelado de datos: Implementación de un modelo estrella (Tablas de Hechos y Dimensiones) para optimizar consultas.
Visualización en Power BI: Creación de un Dashboard interactivo con 5 KPIs estratégicos sobre el catálogo global.
Documentación Técnica: Reporte detallado que incluye topología de red y arquitectura de la base de datos.
Presentación Ejecutiva: Diapositivas con los hallazgos clave para la toma de decisiones.
